# 🤖 Zerocoder Onboarding Bot

AI-ассистент для онбординга новых методистов и стажёров Zerocoder.

## Что делает

- Отвечает на вопросы о структуре команды, инструментах и процессах
- Помогает найти нужные документы и контакты
- Сопровождает новичка в течение первого месяца работы

## Структура онбординга

| Неделя | Фокус | Задачи |
|--------|-------|--------|
| 1 | Ориентация | Знакомство с командой, инструментами, регламентами |
| 2 | Наблюдение | Участие во встречах, первые небольшие задачи |
| 3 | Практика | Самостоятельные задачи по шаблонам |
| 4 | Самостоятельность | Полноценная работа, ретроспектива |

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/YOUR_USERNAME/zerocoder-onboarding-bot.git
cd zerocoder-onboarding-bot
```

### 2. Установить зависимости
```bash
pip install python-telegram-bot
```

### 3. Получить токен бота
Открыть [@BotFather](https://t.me/BotFather) в Telegram → `/newbot` → скопировать токен

### 4. Добавить токен
```bash
# Windows PowerShell
$env:TELEGRAM_BOT_TOKEN = "your_token_here"
```

### 5. Запустить
```bash
python bot.py
```

## Команды бота

- `/start` — начать работу с ботом

## Технологии

- Python 3.14
- python-telegram-bot 22.8
